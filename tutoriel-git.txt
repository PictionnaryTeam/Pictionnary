Clonage :
---------

Le clonage est à faire une seule fois.

git clone <url> <dossier local>

Pour ce projet :
git clone https://github.com/PictionnaryTeam/Pictionnary.git <un dossier de votre choix>


Dire à GIT dans quel dossier l'on travaille :
---------------------------------------------

Indiquer dans quel repository on travaille, c'est à dire le dossier dans lequel vous avez cloné le git pour la première fois.

cd <chemin du dossier qui contient le .git>


Voir l'état du git :
--------------------

Permet de savoir quels fichiers ont été modifiés/ajoutés depuis le dernier pull.

git status


Pour ajouter un fichier a un commit :
-------------------------------------

Une fois que vous avez ajouté ou modifié un fichier, il faut valider les modifications.

Pour ajouter un fichier au commit (commit = ensemble des modifications qui n'ont pas encore été push)
git add <nom du fichier>

Cela fonctionne également avec les dossiers
git add <nom du dossier>


Pour supprimer un fichier :
----------------------------

git rm <nom du fichier/du dossier>


Commit (valider les modifications) :
------------------------------------

git commit -m "message du commit"


Pour télécharger ou upload les modifications :
----------------------------------------------

Tirer les modif
git pull

Pousser les modif (après avoir commit)
git push 


Notes :
-------

/!\ Attention aux espaces dans les noms de fichiers ou de dossiers /!\
